# GlimmerWorks

English | [简体中文](README_zh_CN.md)

Welcome to the GlimmerWorks open-source repository! We aim to build a player-centric open creation platform that inspires creativity and unlocks boundless imagination. No matter where you are in the world, you are welcome to join us in building an open sandbox world together.

### Implemented Features

1. Grid-based procedural world generation powered by chunks and world seeds.
2. Easily moddable and extensible data pack & resource pack system.
3. In-game built-in console.

### Gameplay

1. Explore procedurally generated grid worlds and mine resources.
2. Build your own shelters and constructions.
