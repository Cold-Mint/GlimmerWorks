# GlimmerWorks

[English](README.md) | 简体中文

欢迎来到 GlimmerWorks 开源代码仓库！我们致力于打造以玩家为核心、激发创作灵感、释放无限想象的开放创作平台。无论你来自世界何处，都欢迎与我们一同共建开放沙盒世界。

### 已实现的功能

1. 基于区块和种子程序生成的网格世界。
2. 易于修改和扩展的数据包，材质包系统。
3. 游戏内置控制台。

### 游戏玩法

1. 探索程序化生成的网格世界，挖掘矿石。
1. 建造房屋。
